# 图标文件说明

Plasmo 框架需要 `assets/icon.png` 文件作为插件图标。

## 如何添加图标

1. 将图标文件命名为 `icon.png`
2. 放置在 `assets/` 目录下
3. 推荐尺寸：至少 128x128 像素，支持 PNG 格式

## 临时解决方案

如果暂时没有图标文件，可以从参考项目复制：
```bash
# 从参考项目复制图标
copy "F:\谷歌插件\xiaohongshu-mutual-like\assets\icon.png" "assets\icon.png"
```

或者使用任何 128x128 或更大的 PNG 图片作为临时图标。
