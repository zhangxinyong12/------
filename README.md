## PDF生成功能

插件已集成PDF生成功能，当用户点击"批量打印商品打包标签"按钮时，会自动生成PDF文件并下载。

### 功能说明

1. **自动PDF生成**：当点击打印按钮后，插件会自动捕获页面内容并生成PDF文件
2. **文件命名**：PDF文件会自动命名为 `打印标签_[时间戳].pdf` 格式
3. **保存位置**：PDF文件会保存到浏览器的默认下载目录

### 使用方法

1. 正常点击"批量打印商品打包标签"按钮
2. 插件会自动检测打印事件
3. PDF文件会自动生成并下载到本地

### 技术实现

- 使用 `html2canvas` 将页面内容转换为图片
- 使用 `jsPDF` 将图片转换为PDF文件
- 通过监听 `beforeprint` 事件来捕获打印内容

### 注意事项

- PDF文件会在打印预览页面出现后自动生成
- 如果页面内容较大，生成PDF可能需要几秒钟时间
- PDF文件的质量取决于页面内容的复杂程度
